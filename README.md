# GitHub Trial

A small HTML/JavaScript demo page.

## What it does
- Shows a simple page with a title, an image, and a button.
- Clicking **Change background** changes the background color of the content area to **pink**.

## Files
- `index.html` — the main page and inline JavaScript.
- `images.webp` — image used by the page.

## Run
Just open `index.html` in your browser.

## Notes
The button triggers the `pressed()` function, which updates the `#body` element styles.
